# BlockStack

Tiny console falling-block game in C#.

## Run

```bash
dotnet run
```

Requires .NET 8 SDK.

## Controls

- `A` - move left
- `D` - move right
- `S` or `Space` - rotate

## Notes

- The game runs in the console.
- Blocks fall on a timer, while input is handled every frame.
- Line clearing is implemented.
- The cursor is hidden during play.

## Current limits

- No score yet
- No level progression yet
- No game over screen yet
